# SI-lidity

## Description

A set of solidity contracts from the SI team with Hardhat 3 Alpha!

## Getting Started

### Prerequisites

Node.js 22

### Installation

Just run

`yarn install`

### Quick Start

Just run

`git submodule update --remote --recursive`

`yarn compile`

### Tests

Just run

`yarn test`

### Submodules

Just updates submodules with:

`git submodule update --remote --recursive`
